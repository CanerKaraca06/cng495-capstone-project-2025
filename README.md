# cng495-capstone-project-2025
This Spring-Boot project is a music sharing and listening platform. 
Technologies used are listed below for easier user experience. 
    Java 17 JDK 17.0.12
    Spring Web
    Spring Data JPA
    Spring Security
    Maven
    PostgresSQL
    Hibernate
    AWS S3
Overall, project structure are organized into packages
This packet structure may be utilized to understand the code
Main packages include 
    Model
    Configuration
    Controller
    Repository
    Service
Settings and dependencies are stored in 
    application.properties
    pom.xml 
Installing dependencies:
    mvn clean install
Running the application
    mvn spring-boot:run
